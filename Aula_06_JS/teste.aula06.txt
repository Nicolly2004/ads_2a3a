-Abre a pasta amarela 
-Cria uma pasta fora do pendrive (disco local c)

-Entra no gitHub, copia o HTTP do repositório.
- Abre o git bash dentro da pasta criada e faz o "git clone http..."

(a pasta clonada irá aparecer dentro da oura que foi criada no disco local c)
(clicar nela e abrir o git bash dentro da pasta clonada, ela já estará sendo controlada pela main)

-Faz o comando "git config ---global user.name "Nicolly Reis", para definir o nome de quem está usando e faz "git config ---global user.email "nicollyro2004@gmail.com"
- Faz o "git log" pra saber quem está usando.

-  se um documento de texto ou qualquer alteração for criada dentro da pasta clonada fazer o git status para ver as alterações feitas

-fazer o "git commit -m "descrição"
-fazer 2x o "git push -u origin main ou master"
- Na segunda vez que o push for feito, ele vai funcionar e ai será necessário criar um token pelo gitHub.
